# Personal Expense Tracker

Прост Python скрипт за проследяване на лични разходи.

## Features
- Добавяне на разходи (дата, категория, сума, бележка)
- Показване на сумарен разход и разход по категории
- Всичко се съхранява в CSV файл (`expenses.csv`)
